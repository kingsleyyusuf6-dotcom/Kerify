Frequently Asked Questions

What is KERIFY?▼
KERIFY is an AI-powered identity verification SaaS platform that helps businesses verify user identities using face matching, ID document scanning, liveness detection, and fraud screening.
How accurate is the face verification?▼
Our AI achieves 99.9% accuracy in face matching, using state-of-the-art deep learning models trained on diverse datasets covering all ethnicities and age groups.
Is my data secure?▼
Absolutely. We use AES-256 encryption at rest and TLS 1.3 in transit. We are SOC 2 Type II certified and GDPR compliant.
Can I integrate via API?▼
Yes! KERIFY offers a comprehensive REST API with SDKs for Python, Node.js, PHP, and Java. Webhook support enables async verification workflows.
